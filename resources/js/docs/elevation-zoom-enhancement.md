# Elevation Zoom Enhancement - ปรับปรุงการแสดงผลเมื่อซูม

## ปัญหาที่แก้ไข

**ปัญหา**: เมื่อซูมเข้าไปสุด ไม่เห็นความแตกต่างของสี
**สาเหตุ**: ความละเอียดไม่เพียงพอและสีไม่แตกต่างกันชัดเจน

## การปรับปรุง

### 1. **เพิ่มความละเอียด**

```typescript
// เก่า: 7x7 grid (49 พื้นที่)
const gridSize = 7;

// ใหม่: 10x10 grid (100 พื้นที่)
const gridSize = 10;
```

### 2. **ปรับปรุงการคำนวณสี**

```typescript
// เก่า: 5 สี
if (normalizedElevation < 0.2)
    return '#0000FF'; // Blue
else if (normalizedElevation < 0.4) return '#00FF00'; // Green
// ...

// ใหม่: 10 สีที่แตกต่างกันชัดเจน
if (normalizedElevation < 0.1)
    return '#0066CC'; // Dark Blue
else if (normalizedElevation < 0.2)
    return '#0088FF'; // Light Blue
else if (normalizedElevation < 0.3) return '#00AA44'; // Dark Green
// ... 10 ระดับสี
```

### 3. **ปรับปรุงการแสดงผล**

```typescript
// เพิ่ม stroke สีขาวเพื่อแยกพื้นที่
strokeColor: '#FFFFFF',
strokeOpacity: 0.9,
strokeWeight: 2,

// เพิ่ม fillOpacity
fillOpacity: 0.7,
```

### 4. **อัปเดต Legend**

- เพิ่มสีใหม่ 10 ระดับ
- แสดงความหมายของแต่ละสี
- แสดงจำนวนพื้นที่ (10×10 = 100 พื้นที่)

## สีแสดงความสูง (10 ระดับ)

1. 🔵 **#0066CC** - ต่ำมาก (พื้นที่ราบ)
2. 🔵 **#0088FF** - ต่ำ
3. 🟢 **#00AA44** - ปานกลางต่ำ
4. 🟢 **#00CC66** - ปานกลาง
5. 🟡 **#88CC00** - สูงปานกลาง
6. 🟡 **#FFCC00** - สูง
7. 🟠 **#FF8800** - สูงมาก
8. 🔴 **#FF4400** - สูงมาก
9. 🔴 **#CC0000** - สูงมาก
10. 🔴 **#880000** - สูงมากที่สุด (ภูเขา)

## ข้อดี

- ✅ **ความละเอียดดีขึ้น**: 100 พื้นที่แทน 49
- ✅ **สีแตกต่างชัดเจน**: 10 ระดับสี
- ✅ **เห็นความแตกต่าง**: เมื่อซูมเข้าไปสุด
- ✅ **Stroke สีขาว**: แยกพื้นที่ชัดเจน
- ✅ **Legend ครบถ้วน**: แสดงทุกสี

## การทดสอบ

1. เปิดเครื่องมือ elevation
2. เลือก "แสดงความสูงต่ำด้วยสี"
3. คลิกปุ่ม "โหลดข้อมูลความสูง"
4. ซูมเข้าไปสุด
5. ตรวจสอบว่ามีความแตกต่างของสีชัดเจน

## ผลลัพธ์ที่คาดหวัง

- พื้นที่สี 100 ช่อง (10×10 grid)
- สีแตกต่างกันชัดเจน 10 ระดับ
- เห็นความแตกต่างเมื่อซูมเข้าไปสุด
- Stroke สีขาวแยกพื้นที่ชัดเจน
- Legend แสดงทุกสี
